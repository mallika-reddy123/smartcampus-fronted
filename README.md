# Smart Campus Management - Frontend

React frontend for Smart Campus Management Platform.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file with:

```env
VITE_API_URL=http://localhost:5000/api
```

## Running the App

Development mode:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Features

- Modern React with Hooks
- React Router for navigation
- Tailwind CSS for styling
- Recharts for data visualization
- Axios for API calls
- Dark/Light theme toggle
- Responsive design
- Toast notifications

## Pages

- **Login** - User authentication
- **Signup** - New user registration
- **Dashboard** - Overview with KPIs and charts
- **Resources** - Browse and search resources
- **Booking** - Smart booking with conflict detection
- **My Bookings** - View and manage bookings
- **Analytics** - Predictive analytics and insights
- **QR Booking** - Quick booking via QR code

## Technologies

- React 18
- React Router v6
- Tailwind CSS
- Recharts
- Axios
- React Icons
- React Hot Toast
- Vite
